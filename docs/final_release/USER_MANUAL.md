# INDUSMIND AI — Enterprise User Manual

## 1. Executive AI Command Center (`/dashboard/executive`)
- **Plant Health Score**: Real-time composite health index aggregated across all plant equipment.
- **Financial Impact ROI**: Direct dollar savings calculated from prevented unplanned outages.
- **Risk Summaries**: Strategic risk breakdown categorized by safety, compliance, and asset integrity.

## 2. Plant Administrator Guide (`/dashboard/admin`, `/dashboard/enterprise`)
- **User Management**: Add plant engineers, assign RBAC roles (`Super Admin`, `Admin`, `Manager`, `Engineer`, `Auditor`, `Technician`, `Viewer`).
- **System Health Center**: Monitor live subsystem pulse (API, DB pool, pgvector search, graph traversal speed).
- **Security Audits**: Inspect failed login logs, permission matrices, and security ratings (A+).

## 3. Maintenance Engineer Guide (`/dashboard/equipment`, `/dashboard/decision`, `/dashboard/copilot`)
- **Predictive Maintenance**: Monitor telemetry graphs (vibration, temp, pressure) and RUL estimations.
- **Decision Intelligence**: Evaluate cost vs. downtime vs. safety trade-off recommendations.
- **AI Copilot**: Ask complex technical questions about plant manuals with page-level citations.

## 4. Operator & Technician Guide (`/dashboard/incidents`, `/dashboard/search`)
- **Incidents & Lessons Learned**: Log equipment failures and search historical incident resolutions.
- **Semantic Search**: Fast hybrid search for drawings, SOP manuals, and part numbers.

## 5. Compliance & Auditor Guide (`/dashboard/compliance`, `/dashboard/audit`)
- **Compliance Intelligence**: Track OSHA 1910.119 / ISO 55000 regulations and pending inspection audits.
- **Audit Trail**: Full system audit log tracking all document uploads, configuration updates, and user access.

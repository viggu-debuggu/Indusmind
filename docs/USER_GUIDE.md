# INDUSMIND AI - User Guide

## Core Features
1. **AI Copilot (`/dashboard/copilot`)**: Ask questions about plant drawings, manuals, and SOPs with interactive citation links.
2. **Equipment & Predictive Maintenance (`/dashboard/equipment`)**: View asset health scores, telemetry graphs, and failure predictions.
3. **Knowledge Twin (`/dashboard/twin`)**: Compare asset health snapshots across operating plants.
4. **Industrial Discovery (`/dashboard/discovery`)**: Uncover hidden pattern relationships and cross-department knowledge gaps.
5. **Executive AI Center (`/dashboard/executive`)**: Track enterprise KPIs, financial impact, and strategic risk summaries.
6. **Enterprise Hub (`/dashboard/enterprise`)**: Real-time system health, performance metrics, security reports, and deployment status.
